# python_max_tcp

пример сгенерированного клиента и логики взаимодействия в `main.py`

## USAGE

через uv установить зависимости

запуск:

### прохождение авторизации

>[!warning]
> не учитывает случай, если после числового кода не просит пароль подтверждения, нужно поправить

Авторизация:

```
uv run main.py auth
```

после авторизации

```
uv run main.py start
```

далее интерактивно проходите авторизацию
- отсутствуют проверки ввода, вводите сразу правильно
  - телефон надо вводить с `+7` кода
  - пароль нужно ввести корректно с первого раза (не заложил attempts логику)


## Баги py реализации

используются для помощи в IDE автокомплите TypedDict аннотации, в messagepack отключен strict режим - проверяйте на фактическом поведении. Можно игнорировать сгенерированные типы и в параметрах использовать обычные dict словари
